# Recurring Task Manager

A small, dependency-free web app to manage recurring tasks.

## Features

- Add recurring tasks with a cadence (days, weeks, months)
- See the next due date for each task
- Mark tasks complete to advance the next due date
- Delete tasks
- Tasks are stored in `localStorage`

## Run locally

Open `index.html` directly in your browser, or run a local static server:

```bash
python -m http.server 8000
```

Then visit <http://localhost:8000>.
